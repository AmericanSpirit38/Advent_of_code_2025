# Advent of Code 2025 — Solutions

This repository contains my solutions to **Advent of Code 2025**.

## File Structure

Each solution file follows this naming convention:

- `<day_number>.py` — Part 1  
- `<day_number>-02.py` — Part 2  

Each puzzle also includes its corresponding input file:

- `<day_number>_input.txt`
